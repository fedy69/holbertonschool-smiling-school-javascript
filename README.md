# Holberton SmileSchool Javascript

## Description

Implement 3 web pages with Bootstrap and javascript (jQuery)
In this project, you will finalize the previous project [0x0B. Implement a design with bootstrap](https://github.com/fedy69/holbertonschool-smiling-school) and make some parts dynamic with Javascript (JQuery exactly).
You will use all HTML/CSS/Accessibility/Responsive design/Bootstrap/Javascript knowledges that you learned previously.
You won’t have a lot of instruction, you are free to implement it the way that you want - the objective is simple: Have fully functional web pages that look the same as the designer file.

![cover](images/cover.jpg)

## Requirements :pushpin:
- You have to use Bootstrap
- Your ```styles.css``` must be as small as you can - **you must use as much as you can Bootstrap classes**
- You have to use JQery
- Your ```scripts.js``` must contain all your Javascript part
- Your Javascript must be executed only when the document is loaded

---




* **Interactions note:**
  * Web pages must switch to the tablet version when the screen width is 768px
  * Web pages must switch to the mobile version when the screen width is 576px
  * button hover/active: opacity: 0.9
